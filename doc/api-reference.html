    Trans

    Trans is split into 2 main components

    Trans.QueryBuilder

    Provides functions to build queries with conditions on translated fields. Take a look at the Trans module documentation to see how you can set up convenience helpers in your model module to avoid repetition of default values

    Trans.Translator

    Provides functions to access translated fiels in a model. Take a look at the Trans module documentation to see how you can set up convenience helpers in your model module to avoid repetition of default values
